# 皮带传送带实验使用指南

## 📦 快速搭建

### 1. 创建场景结构

```
BeltConveyorExperiment (根对象)
├── Conveyor (Cube) - 传送带
│   ├── BoxCollider
│   └── BeltConveyor (脚本)
├── Block (Cube) - 物块
│   ├── Rigidbody
│   ├── BoxCollider
│   ├── PhysicsObject (脚本)
│   ├── DataLogger (脚本)
│   └── EnergyCalculator (脚本)
├── ParameterController (空对象)
│   └── ParameterController (脚本)
└── ExperimentUI (Canvas) - UI界面
    ├── StartButton / PauseButton / ResetButton / StopButton
    ├── FrictionSlider / AngleSlider / BeltSpeedSlider
    └── 显示Text组件
```

### 2. 配置传送带

**创建：** 右键 → 3D Object → Cube

**Transform：**
- Position: (0, 0, 0)
- Rotation: (0, 0, 0)
- Scale: (5, 0.1, 10)

**Inspector设置：**
- Belt Speed: 2 m/s
- Belt Width: 5 m
- Belt Length: 10 m
- Is Running: ☑️
- Incline Angle: 0-90° (角度制)

### 3. 配置物块

**创建：** 右键 → 3D Object → Cube

**Transform：**
- Position: (0, 0.3, -4)
- Scale: (1, 1, 1)

**Rigidbody：**
- Mass: 1 kg
- Use Gravity: ☑️
- Is Kinematic: ☐
- Drag: 0.1

**PhysicsObject：**
- Mass: 1
- Object Color: 红色
- **Is Particle**: ☐ (启用质点模式)

**质点模式说明：**
- ☑️ Is Particle = 物体锁定旋转，忽略形状尺寸
- Particle Collider Size: 0.01 (极小碰撞体)
- Particle Visual Size: 0.1 (可视化小球)

### 4. 连接UI

**创建Canvas：** 右键 → UI → Canvas

**添加按钮和滑块：**
- StartButton / PauseButton / ResetButton / StopButton
- FrictionSlider (0-1) / AngleSlider (0-90) / BeltSpeedSlider (0-10)
- Text组件显示实时数据

**ParameterController连接：**
- 拖入所有Slider和Text对象

## 🎯 核心功能

### 1. 倾斜实验 (mgsinα = μmgcosα)

**设置倾斜角度：**
```
Incline Angle = 30°
```

**自动计算（保留2位小数）：**
- mgsinα = 4.91 N (重力沿斜面向下)
- mgcosα = 8.50 N (重力垂直斜面)
- μ临界 = tan30° = 0.58

**判断：**
- μ > 0.58: 物体保持平衡 ☑️
- μ = 0.58: 临界状态 ⚠️
- μ < 0.58: 物体开始下滑 ❌

**查看计算：** 右键BeltConveyor → 显示物理计算信息

### 2. 质点模式

**启用方法：**
1. 选择Block对象
2. 勾选 PhysicsObject → Is Particle

**效果：**
- ✅ 锁定旋转
- ✅ 碰撞体极小化
- ✅ 忽略形状和尺寸
- ✅ 适用于理论公式验证

### 3. 数据记录

**DataLogger自动记录：**
- 时间、速度、位置
- 动能、势能
- 理论值对比

**导出数据：** 调用ExportToJSON()

## 📊 物理公式

| 公式 | 说明 |
|------|------|
| a = g·sin(θ) - μ·g·cos(θ) | 斜面加速度 |
| E_k = ½·m·v² | 动能 |
| E_p = m·g·h | 势能 |
| μ临界 = tan(θ) | 临界摩擦系数 |

## ❓ 常见问题

**Q: 物块不移动？**
- 检查Rigidbody的Is Kinematic是否为false
- 检查传送带Is Running是否为true
- 检查摩擦系数设置

**Q: 图表不显示？**
- 检查SimpleChartDrawer配置
- 检查数据记录是否开始
- 检查Max Points设置

**Q: 三角函数值显示？**
- 所有值统一保留2位小数
- Inspector和Console都显示格式化值
