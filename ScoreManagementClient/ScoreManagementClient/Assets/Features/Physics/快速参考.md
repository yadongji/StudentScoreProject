## 🎯 核心物理公式

### 斜面运动
```
加速度：a = g·sin(θ) - μ·g·cos(θ)
速度：v = v₀ + a·t
```

### 能量
```
动能：E_k = ½·m·v²
势能：E_p = m·g·h
弹性势能：E_e = ½·k·x²
机械能：E_total = E_k + E_p + E_e
```

### 皮带传动
```
线速度：v = ω·r
传动：ω_driven = ω_driver × (r_driver/r_driven) × ratio
```

## 📊 数据采集流程

```
FixedUpdate (每0.02s)
    ↓
记录：Time.time, velocity, position
    ↓
计算：speed, KE, PE, E_total
    ↓
存入队列（最大500个点）
    ↓
SimpleChartDrawer绘制曲线
    ↓
显示实时数据
```

## 🐛 常见问题速查

**Q: 物块不移动？**
- 检查Rigidbody的Is Kinematic = false
- 检查传送带Is Running = true
- 检查摩擦系数是否合理

**Q: 图表不显示？**
- 检查SimpleChartDrawer配置
- 检查数据记录是否开始
- 检查Max Points设置

**Q: 能量计算不对？**
- 检查Gravity设置（9.81）
- 检查Reference Height
- 检查物体位置

**Q: 理论值偏差大？**
- 物理引擎数值精度
- Fixed Timestep设置
- 摩擦模型简化
